Available controls:

switch icon

Switch

Click to toggle on or off. Sends an event whenever the value changes.

Returns value - boolean

Supported on iOS, Android, and computers.

accelerometer icon

Accelerometer

Only available on devices with accelerometers. Sends an event whenever the device orientation changes.

Returns x-value - float 0 to 1, y-value: float 0 to 1

Supported on iOS only. Not available on Android devices or computers

touchpad icon

Touchpad

Touch or click and drag. Sends data whenever a touch or click event occurs.

Returns x-value - float 0 to 1, y-value: float 0 to 1

Touches supported on iOS and Android. Click-and-drag to mimic touches on computers.
